### Необходимо установить [Node.js](https://nodejs.org/en/download/).
### Затем нужно устанавить Vue.js
```bash
  npm i -g @vue/cli
```
## Запуск проекта
### 1.Серверная часть
Из корневой папки проекта:
```bash
  npm ci
```
#### Запуск сервера:
```bash
  npm start
```
### 2.Клиентская часть
 Из корневой папки проекта:
 ```bash
  cd client 
  npm ci
 ```
#### Запуск клиентской части: 
 ```bash
  npm run serve
 ```
